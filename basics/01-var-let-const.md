# Difference Between var, let, and const

## var

- Function scoped
- Can be redeclared
- Can be reassigned
- Hoisted

Example:

var a = 10
var a = 20

## let

- Block scoped
- Cannot be redeclared
- Can be reassigned

Example:

let a = 10
a = 20

## const

- Block scoped
- Cannot be redeclared
- Cannot be reassigned

Example:

const a = 10
